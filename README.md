📌 Project Overview

This is a simple Inventory Management System built using HTML, CSS, and JavaScript.
It allows users to manage product/items efficiently by adding, updating, and deleting inventory records in a clean UI.

🛠 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
✨ Features
➕ Add new inventory items
✏️ Update existing items
❌ Delete items
📋 View all inventory in a structured list/table
🎨 Clean and responsive UI design
⚡ Fast and lightweight (no frameworks)
📂 Project Structure
index.html        → Main HTML file
style.css         → Styling file
script.js         → JavaScript logic
assets/           → Images or static files (if any)
